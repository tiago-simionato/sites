RESUMO DE TAGS HTML E PROPRIEDADES DE CSS's

Tags HTML

  • p

    Cria um parágrafo

  • h1 até h6

    Cria títulos para organizar seu conteúdo

  • img

    Carrega imagens no site

  • video

    Carrega vídeos no site

  • audio

    Carrega audios no site

  • iframe

    cria algo que ainda não sei o que é

  • a

    É uma âncora que cria ligações entre páginas

  • ol

    Cria listas ordenadas

  • ul

    Cria listas onde a ordem não é importante (como esta aqui)

  • li

    Tag usada para cada elemento de uma lista (com ordem ou sem)

  • strong

    Tag para negrito (prefira usar CSS)

  • em

    Tag para itálico (emfase) (prefira usar CSS)

  • ins

    Tag para underline

  • sub

    Tag para coisas subescritas

  • sup

    Tag para coisas sobrescritas

  • mark

    Tag para marca-texto

  • strike

    Tag para riscar o texto

  • q

    Usado em citações (quotes)

  • blockquote

    Use em citações em bloco

  • abbr

    Use para criar janelinhas que abrem ao passar o mouse sobre o conteúdo

  • code

    Use em códigos fontes

  • bdo

    Inverte o Texto

  • table

    principal tag usada para criar tabelas

  • tr

    table row, cria uma linha para a tabela

  • td

    table data, usado para colocar os dados dentro de uma linha (cada td delimita uma coluna)

  • thead

    cabeçalho de uma tabela grande

  • th

    Usado para adicionar os dados nos cabeçalhos de tabelas

  • tbody

    parte principal de uma tabela grande

  • tfoot

    rodapé de uma tabela grande

  • caption

    Tag onde se coloca a descrição de uma tabela

  • colgroup

    É uma tag que serve para facilitar a formatação de colunas específicas de uma tabela. Dentro dela crie uma tag col para cada coluna que a tabela tiver

  • scope

    Não é uma tag, mas uma característica para ser adicionada nas tags de tabelas. Ela ajuda os mecanismos de busca a analisar as tabelas do site, portanto não gera resultados visuais.

  • colspan

    Não é uma tag, mas uma característica para ser adicionada nas tags de tabelas. Ela faz com que uma linha seja expandida para ocupar mais colunas

  • rowspan

    Não é uma tag, mas uma característica para ser adicionada nas tags de tabelas. Ela faz com que uma coluna seja expandida para ocupar mais linhas

Tags de Agrupamento

  • div

    Cria uma caixa box level

  • span

    Cria uma caixa do tipo inline level

  • header

    é uma div com a função semântica de ser o cabeçalho do site

  • nav

    É uma div com a função semântica de ser um menu

  • main

    É uma div com a função semântica de ser o conteúdo principal do site

  • section

    É uma div que representa uma seção genérica do conteúdo principal

  • article
  • aside
  • footer

    É uma div com a função semântica de ser o rodapé do site

Formatadores de CSS's

  • font-size

    Ajusta o tamanho do corpo da fonte

  • font-family

    Ajusta o estilo da fonte

  • font-style

    Coloca em italico ou normal

  • font-weight

    Ajusta a 'grossura' da fonte (bold)

  • text-align

    alinha o texto no meio, na esquerda, na direita, ou justificado

  • text-decoration

    pode sublinhar o texto, entre outras coisas

  • color

    Arruma a cor do elemento em questão

  • background-color

    Muda a cor do fundo

  • height

    Ajusta a altura da caixa

  • width

    Ajusta a largura da caixa

  • display

    Configura como que a caixa será exibida na tela

    Tipos de display: none: esconde o conteudo, inline-block: coloca a caixa como inline level, block: coloca a caixa como box level)

  • font (shorthand)

    Shorthand para ajuste de fontes (style, weight, size, family)

  • border (shorthand)

    Shorthand para ajuste de bordas (largura, tipo, cor)

    tipos de borda: solid, dashed, dotted, double, groove

  • padding (shorthand)

    Cria um afastamento entre a borda de uma caixa e seu interior, seu conteudo

    shorthand(top, right, bottom, left)

  • margin (shorthand)

    Cria um afastamento entre a borda de uma caixa e seu exterior

    shorthand(top, right, bottom, left)

  • outline (shorthand)

    Cria uma borda depois da borda, mas dentro da margem

    shorthand(largura, tipo, cor)

  • vertical-align

    alinha caixas verticalmente (pode ser usado para alinha td em tabelas)

  • box-shadow (shorthand)

    colocar sombras em caixas

    shorthand:(offsetx, offsety, espalhamento, cor)

  • border-radius

    Usado para arredondar os vertices da caixa

  • border-image (shorthand)

    usado para criar a borda da caixa a partir de um arquivo de imagem

    shorthand:(source, slice, repeat)

  • background-size

    Usado para ajustar o tamanho da imagem de fundo que for carregada na página. Valores interessantes: cover; contain

  • background-repeat

    Usado para definir se a imagem de fundo deverá ou não se repetir e em quais eixos (x, y)

  • background-attachment

    Pode ser usado para definir se a imagem irá rolar com ou fundo ou ficará fixa

  • background (shorthand)

    (color -> image -> position -> repeat -> size -> attachment) (pode ser que o size não funcione)

  • position

    pode ser setada em relative ou absolute. Por padrão as caixas já vem com posições relativas. Se setada em absoluto é possível configurar as características left e top da caixa. Outro valor interessante dessa regra é o sticky, que pode ser usado junto com as regras top e left para manter o elemento na tela mesmo após sair da rolagem

  • left

    Aplica uma translação na borda superior esquerda da caixa com relação à parte esquerda da tela

  • top

    Aplica uma translação na borda superior esquerda da caixa com relação à parte superior da tela

  • transform

    Pode ser usado com algumas funções para aplicar mudanças uteis nas caixas

    Útial para centralizar duas caixas. colocando left:50%;top:50%;transform:translate(-50%,-50%); na caixa interior ela ficará alinhada a caixa pai

  • font-variant

    util para deixar o texto em uma espécie de "semi-caps-lock"

  • border-collapse

    usado para juntar as linhas e colunas de uma tabela

  • overflow

    formatador útil para lidar com conteúdos que criam uma rolagem lateral em conteúdos exibidos em telas pequenas

Pseudo Classes CSS

pseudo classes são usadas para mudar a formatação do conteudo depedendo de seu estado

  • :hover

    Ao Passar o Mouse Por cima

  • :visited

    site ja visitado

  • :active

    ativa quando o elemento esta sendo clicado

  • :root

    Usado para colocar variaveis no começo do arquivo CSS

  • :nth-child()

    Aplica certas configurações para os filhos de algum elemento. O argumento entre parentese indica a cada quantas células a configuração será a aplicada

Pseudo Elemento

pseudos elementos são usados para alterar o conteudo do html

  • ::after

    usado para adicionar um conteudo depois de cada elemento de algum tipo

Seletores Personalizados

  • #

    indica que é id

  • .

    indica que é classe

  • :

    indica que é pseudo classe

  • ::

    indica que é pseudo elemento

  • >

    indica que é um elemento filho de outro

  • *

    Seletor global

Funções CSS

  • url()

    Serve para indicar onde que um arquivo pode ser encontrado. Pode ser na web ou local. Util quando queremos usar imagens de fundo ou outros arquivos.
