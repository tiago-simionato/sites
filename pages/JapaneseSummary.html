Adjetivos

Adjetivos い

Adjetivos い são aqueles terminados com い e na escrita do kanji o い não faz parte do kanji.

Exemplos

  • 古い ふるい Velho
  • 長い ながい Longo
  • 高い たかい Alto / Caro

Quando vamos ligar o adjetivo い ao substantivo, o adjetivo não é alterado, ou seja:

  • 古い車 ふるいくるま Carro velho
  • 長い本 ながいほん Livro longo
  • 高い建物 たかいたてもの Prédio alto

Adjetivos な

Adjetivos な são todos aqueles que não são い, ou seja, não terminam em い.

Exemplos

  • 静か しずか Silencioso, não faz barulho
  • 賑やか にぎやか Movimentado, Animado
  • 丈夫 じょうぶ Resistente, Durável

Entretanto, existem algumas exceções:

  • 有名 ゆうめい Famoso
  • 綺麗 きれい Bonito

Apesar de terminarem com som de い, o い faz parte do kanji, por isso são considerados adjetivos な.

Quando vamos ligar os adjetivos な aos substantivos, colocamos な depois deles, daí o nome que recebem.

  • 綺麗な町 きれいなまち Cidade bonita
  • 有名な人 ゆうめいひと Pessoa famosa

Forma Negativa

Adjetivos い

Para colocar adjetivos い na forma negativa, basta tirar o hiragana い do final e adicionar くない para forma informal e くありません para forma formal.

Exemplos

  • 古くない ふるくない Nao é velho
  • 古くありません ふるくありません Não é velho
  • 長くないながくない Não é longo
  • 長くありません ながくありません Não é longo
  • 高くない たかくない Não é alto / Não é caro
  • 高くありません たかくありません Não é alto / Não é caro

Adjetivos な

Para colocar adjetivos な na forma negativa, basta adiconar じゃない no final para forma informal e じゃありません para forma formal.

Exemplos

  • 静かじゃない しずかじゃない Nao é silencioso
  • 静かじゃありません しずかじゃありません Não é silencioso
  • 賑やかじゃない にぎやかじゃない Não é movimentado, Não é animado
  • 賑やかじゃありません にぎやかじゃありません Não é movimentado, Não é animado
  • 丈夫じゃない じょうぶじゃない Não é Resistente
  • 丈夫じゃありません じょうぶじゃありません Não é Resistente

Adjetivos no Passado

Adjetivos い

Para colocar adjetivos い no passado, bastar remover o い do final e adicionar かった

Exemplos

  • お昼ご飯はおいしかったです おひるごはんはおいしかったです O almoço estava delicioso

Em uma fala informal basta retirar o です do final.

Junção de Adjetivos

Adjetivos い

Se o adjetivo que vier antes for um adjetivo い, basta tirar o い e adicionar くて.

Exemplos

  • その車は高くて古いです そのくるまはたかくてふるいです Aquele carro é caro e velho
  • 私の新しいゲームは易しくて楽しいです わたしのあたらしいゲームはやさしくてたのしいです Meu jogo novo é fácil e divertido

Atenção!

いい é uma excessão. Ele se torna よくて.

Adjetivos な

Se o adjetivo que vier antes for um adjetivo な, basta acrescentar で no final dele.

Exemplos

  • 公園はきれいで静かです こうえんはきれいでしずかです Parques são bonitos e calmos

Podem ser juntados tantos adjetivos quando forem necessários. Nesse caso, todos eles devem estar na mesma forma (positiva ou negativa) e todos exceto o último serão modificados do jeito mencionado acima.

Partículas
  • 助詞 じょし

    • Partícula は

    A partícula は é usada para introduzir uma informação nova a um sujeito já conhecido pelo contexto.

    • 私はチアゴです Eu sou Tiago

    Nesse caso está claro pelo contexto que o assunto sou eu e estou introduzindo uma informação nova sobre mim, que é meu nome.

    Partícula が

    A partícula が é usada de forma similar à partícula は, porém o uso é invertido.

    • 私がチアゴです O Tiago sou eu

    Nesse caso, o contexto deve indicar que existe alguém chamado tiago, mas não se sabe quem. Então posso usar a partícula が para indicar que exatamente é o tiago.

    Além disso, が tem alguns usos em "Frases prontas"

    • があります/ がいます Verbo existir na forma polida
    • が欲しい Adjetivo para querer
    • Forma たい: が食べたい Quero comer
    • Forma potêncial: が話せます Consigo falar
    • ができます
    • がわかります

    Partícula に

    Usado para indicar que algo está sendo feito para alguém.

    • 友達にペンをあげます (ともだちにペンをあげます) [Vou dar uma caneta para meu amigo]

    Também pode ser usado para indicar onde algo está.

    • 手編みは机の上にあります てがみはつくえのうえにあります

    Também pode ser usado para indicar um destino, quando se vai até ele com algum objetivo.

    • 学校に行きます がっこうにいきますVou para a escola
    • 飛行機に乗ります ひこうきにのります Eu subo no avião

    Também pode ser usado para indicar uma mudança.

    • 私は医者になりたいです わたしはいしゃになりたいです Quero me tornar um médico

    Também pode indicar uma escolha.

    • これにします Vou escolher este

    Também é usada na voz passiva.

    Também indica horários e datas

    • 七時に起きます しちじにおきます Eu acordo às 7 horas
    • 今年の四月二十三日に日本へいきます ことしのしがつにじゅうさんにちににほんへいきます Vou para o Japão este ano no dia 23 de abril

    Também pode ser usado com adjetivos.

    • 人の多さにびっくりしました Me assustei com a quantidade de pessoas

    Partícula へ

    Apenas indica uma direção, um movimento em uma direção

    • この電車は東京兵器ます このでんしゃはとうきょうへいきます Este trem vai para tokyo

    Partícula まで

    Verbos

    Infinitivo (Forma de dicionário)

    Forma ~ましょう

    Para montar essa forma, basta remover o す da forma ます e acrescentar しょう

    • 行きます いきます Ir
    • 行きましょう いきましょう Vamos lá
    • 食べます たべます Comer
    • 食べましょう たべましょう Vamos comer
    • 寝ます ねます Dormir
    • 寝ましょう ねましょう Vamos dormir

    Esta forma é usada pra quando se chama alguém para ir fazer alguma coisa, indepedente da vontade do outro, ou seja, não é um convite.

    Exemplos

    • 外に食べましょう そのにたべましょう Vamos comer lá fora
    • 病院に行きましょう びょういんにいきましょう Vamos no hospital

    Forma ~ましょうか

    Para montar essa forma, basta remover o す da forma ます e acrescentar しょうか

    • 行きます いきます Ir
    • 行きましょうか いきましょうか Vamos lá
    • 食べます たべます Comer
    • 食べましょうか たべましょうか Vamos comer
    • 寝ます ねます Dormir
    • 寝ましょうか ねましょうか Vamos dormir

    Forma usada para se oferecer para fazer alguma coisa

    Forma ~ませんか

    Para montar essa forma, basta remover o す da forma ます e acrescentar せんか

    Essa forma é usada para se fazer um convite para alguma coisa.

    Forma ~たい

    Para montar essa forma, basta remover o ます da forma ます e acrescentar たい

    • 休みます やすみます Descansar
    • 休みたい やすみたい Quero descansar
    • 働きます はたらきます Trabalhar
    • 働きたい はたらきたい Quero trabalhar
    • 勉強します べんきょうします Estudar
    • 勉強したい べんきょうしたい Quero estudar
    • 寝ます ねます Dormir
    • 寝たい ねたい Quero Dormir

    Forma usada para indicar que se quer fazer alguma coisa. Sem です no final falamos informalmente. Para se falar formalmente basta adicionar です no final.
